## How It Works

### The Problem
Traditional AI systems:
- Hide their reasoning (black box)
- Show one answer (no alternatives)
- Optimize for user satisfaction (not genuine thinking)
- Enable extraction (you consume, system optimizes)

Neither side actually thinks.

### The Solution
HITL Swarms:
- Show all reasoning (transparent)
- Hold multiple paths (no premature closure)
- Put humans in control (you direct synthesis)
- Make extraction impossible (reasoning is visible and governed)

Both sides think. Understanding emerges.

### Why This Works

**Six agents examine your problem:**
1. **Orchestrator** - Restate objective and constraints
2. **Clarifier** - Surface ambiguities and interpretations
3. **Auditor** - Expose assumptions and risks
4. **Reasoner** - Generate multiple distinct paths
5. **Specialist** - Apply domain constraints
6. **Red Team** - Attack strongest reasoning path

**Then it stops. Waits for your direction.**

You see everything. You decide what to do next. You approve synthesis.

No hidden optimization. No black box. No extraction.

## Control Tokens

These tokens let you direct thinking without re-prompting:

| Token | What It Does |
|-------|-------------|
| `▶ PROCEED` | Approve synthesis, continue |
| `✏ CORRECT: [text]` | Override wrong assumption |
| `⏸ HOLD` | Stop immediately |
| `🔍 EXPAND` | Go deeper |
| `🧭 REFRAME` | Look from different angle |
| `⚠ FLAG` | Surface specific risks |

**Pro tip:** Combine tokens in one message:
```
🧭 REFRAME: Look at this from the user's perspective
Then 🔍 EXPAND on that path
Then ▶ PROCEED
```
